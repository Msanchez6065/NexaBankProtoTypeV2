# NexaBank Prototype

A modern, interactive banking application prototype built with HTML, CSS, and JavaScript. Features a complete banking dashboard with transfers, bill payments, loan management, and an AI-powered chat assistant.

## Features

- **Secure Login**: Multi-factor authentication with SMS verification
- **Dashboard**: Account balances, recent transactions, and financial overview
- **Money Transfers**: Between accounts or to other NexaBank users
- **Bill Payments**: Pay pending bills or custom payments
- **Card Management**: Freeze/unfreeze debit and credit cards
- **Loan Management**: View balances, make payments, track progress
- **Transaction History**: Complete activity log
- **AI Chat Assistant**: Natural language commands for banking operations
- **Responsive Design**: Works on desktop and mobile devices

- **Vanilla JavaScript**: No frameworks, pure JS for interactivity
